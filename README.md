# mIFology

Le projet de Web sémantique pour rassembler des données sur des divinités grecques.

## Getting Started
Pour lancer le projet, tapez la commande "npm start" depuis le répertoire racine.

### Prerequisites

Aucun prérequis.

### Installing

Aucune installation.

## Built With

* HTML
* Javascript
* JQuery
* Node
