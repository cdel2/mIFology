# mIFology

Le projet de Web sémantique pour rassembler des données sur des divinités grecques.

## Getting Started

Pour tester le projet il suffit d'ouvrir index.html dans un navigateur avec la console.

### Prerequisites

Aucun prérequis.

### Installing

Aucune installation.
